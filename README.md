# PPO Maze Navigation

Project overview, setup instructions

## Setup

```bash
pip install -r requirements.txt
```

## Project Structure

- `src/` - Source code
- `configs/` - Configuration files
- `notebooks/` - Jupyter notebooks
- `models/` - Trained models
- `logs/` - Training logs
- `results/` - Experimental results
- `figures/` - Generated figures
- `videos/` - Demo videos
